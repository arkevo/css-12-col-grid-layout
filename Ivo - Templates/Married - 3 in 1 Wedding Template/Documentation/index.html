General Markup

The general template structure is the same throughout each of the 3 design options. They are "Steven and Anne", "Jessica and Joe", and "Max and Alexandra". Simply choose the folder matching the design you want and edit how you need.

  1. Choose a main color (inside css/colors folder) and include it regularly AFTER the main "style.css" file

  2. Remove the rest of the style switcher code before production, HTML and CSS files.

Navigation

The navigation is very easy to customize, a link must have an "href" and "data-id" matching a section with the same ID on the page. As well as the class ".scroll-link"

A dropdown in structured simply as well:

<li><a href="#registry" data-id="registry" class="scroll-link">Registry</a></li>
<li class="nav-btn"><btn data-toggle="modal" data-target="#rsvp-modal" class="btn btn-primary btn-lg">RSVP</btn></li>
<li class="dropdown">
  <a href="#" class="dropdown-toggle" data-toggle="dropdown" role="button" aria-haspopup="true" aria-expanded="false"><span class="glyphicon glyphicon-menu-down"></span></a>
  <ul class="dropdown-menu">
    <li><a href="components.html">Components</a></li>
    <li><a href="../index.html">Demos</a></li>
  </ul>
</li>

Animations

The animations on the page are easy to reuse anywhere by simply applying the animation you want as a class (.rightReveal for example) on the element you want to animate. It uses scrollReveal.js.

/* ---- animations ---- */
if (typeof sr == 'undefined') {
    window.sr = ScrollReveal({
        duration: 1500,
        delay: 50
    });
}
Royal_Preloader.config({
    onComplete: function () {
        triggerReveals();
    }
});
function triggerReveals() {
    sr.reveal('.bottomReveal', {
        origin: 'bottom'
    }).reveal('.leftReveal', {
        origin: 'left'
    }).reveal('.rightReveal', {
        origin: 'right'
    }).reveal('.topReveal', {
        origin: 'top'
    });

    sr.reveal('.rotateBottomReveal', {
        origin: 'bottom',
        rotate: { x: 90 }
    }).reveal('.rotateLeftReveal', {
        origin: 'left',
        rotate: { y: 90 }
    }).reveal('.rotateRightReveal', {
        origin: 'right',
        rotate: { x: 90 }
    }).reveal('.rotateTopReveal', {
        origin: 'top',
        rotate: { y: 90 }
    })

    sr.reveal('.scaleReveal', {
        origin: 'top',
        scale: 0.6
    });
}

JavaScript JavaScript customization and highlights

Magnific Popup Lightbox

This is the js activating our image, video, and gallery zoom. Simply add the corresponding class to your a tag and the link to your image or video as the href. It works the same way for videos, maps, and galleries as well.

/* ---- gallery ---- */
$('#gallery').magnificPopup({
    delegate: 'a.zoom',
    type: 'image',
    fixedContentPos: false,
    removalDelay: 300,
    mainClass: 'mfp-fade',
    gallery: {
        enabled: true,
        preload: [0,2]
    }
});

/* ---- youtube, vimeo, map popup ---- */
$('.popup-youtube, .popup-vimeo, .popup-gmaps').magnificPopup({
  disableOn: 700,
  type: 'iframe',
  mainClass: 'mfp-fade',
  removalDelay: 160,
  preloader: false,
  fixedContentPos: false
});

Parallax.js code

To easily add a parallax effect behind an element, add data-parallax="scroll" to the element you want to use, and specify an image with data-image-src="/path/to/image.jpg".

<div class="parallax-window" data-parallax="scroll" data-image-src="/path/to/image.jpg"></div>
